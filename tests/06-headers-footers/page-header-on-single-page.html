Here is a small document, showing how to use the "pageheader" class to declare page header. There is nothing complicated, "pageheader" are normal <div> rendered as usual.

By default, a "pageheader" is visible only on print media, except if the class "visible" is applied :
  • On page #2, the header is not visible in the browser, only in the print media.
  • On page #3, the header is visible both in the browser and the print media.

On the generated PDF, the position of every "This is page #..." frame must be identical on all pages. The "pageheader" SHALL NOT disturb the position of elements.

This is a header on page 2 (invisible in the browser)
Above, an invisible pageheader have been inserted, using the following HTML code : <div class="pageheader"> This is a header on page 2 </div>

This is page #2.

This is a header on page 3 (visible in the browser)
Above, a visible pageheader have been inserted, using the following HTML code : <div class="pageheader visible"> This is a header on page 3 </div>
