Page footers work well only on single page. Because we need to positionate them at the bottom of the page. (Only .single-page has a fixed height.)
Here is a small document, showing how to use the "pageheader" class to declare page header. There is nothing complicated, "pageheader" are normal <div> rendered as usual.

By default, a "pageheader" is visible only on print media, except if the class "visible" is applied :
  • On page #2, the header is not visible in the browser, only in the print media.
  • On page #3, the header is visible both in the browser and the print media.

On the generated PDF, the position of every "This is page #..." frame must be identical on all pages. The "pagefooter" SHALL NOT disturb the position of elements.

To insert an invisible footer, use the following HTML code : <div class="pagefooter"> This is a footer on page 1 </div>

To insert a visible footer, use the following HTML code : <div class="pagefooter visible"> This is a footer on page 2 </div>
