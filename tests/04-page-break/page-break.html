This document contains ONE .page element. In the browser, it appears as a very long continuous page. But using page-break, once printed this document will contain 5 physical pages (separated by 4 different flavors of page-breaks).

On the generated PDF, the position of every "This is page #..." frame must be identical on all pages. The "pagebreak" SHALL NOT disturb the position of elements.

All page-breaks are invisible on print jobs. For "wysiwyg" purpose, some are visible in the browser. Here are the 4 different flavors:
  • invisible [between pages #1 and #2]
  • visible [between pages #2 and #3]
  • visible + margin-left (label in the left side) [between pages #3 and #4]
  • visible + margin-left (label in the right side) [between pages #4 and #5]

This is page #1.

To insert an invisible pagebreak, use the following HTML code : <div class="pagebreak"></div>
(You cannot see this invisible one from your browser)

This is page #2.

To insert an visible pagebreak, use the following HTML code : <div class="pagebreak visible"></div>

This is page #3.

To insert an visible pagebreak, with text in the left margin, use the following HTML code : <div class="pagebreak visible margin-left"></div>

This is page #4.

To insert an visible pagebreak, with text in the right margin, use the following HTML code : <div class="pagebreak visible margin-right"></div>

This is page #5.
