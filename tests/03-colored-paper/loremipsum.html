If your work is going to be printed on colored paper, for "wysiwyg" purpose you can add a specific color in a small <style> tag at the beginning of your document. Using @media screen, the background-color won't be present on the print job, or the printed PDF.
@media screen { .page { background-color: rgb(255,255,192); } } (avoid alpha color if you don't want to see the stripped 'body')
