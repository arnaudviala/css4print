If your work is going to be printed on color paper, you can simulate the result in your browser by specifying a background color that would be visible only on screen (not on print jobs).
Simply add the following CSS rule at the beginning of your document. Using @media screen, the background-color won't be present on the print job, or the generated PDF.
@media screen { .page { background-color: rgb(255,255,192); } } (avoid alpha color if you don't want to see the stripped 'body')
